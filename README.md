# 📝 Todo List

Projeto simples de Lista de Tarefas desenvolvido para praticar frontend utilizando HTML, CSS e JavaScript.

---

## 🚀 Funcionalidades

✅ Adicionar tarefas  
✅ Marcar tarefas como concluídas  
✅ Remover tarefas  
✅ Interface moderna e responsiva  

---

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
